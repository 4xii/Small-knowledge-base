## webpack.config.js配置
(```)
  const path = require('path');

  module.exports = {
    //入口
    entry: './src/index.js',
    output: {
      //虚拟打包路径，文件夹不会真正生成，而是再8080虚拟生成
      publicPath: 'xuni',
      //打包出来的文件名
      filename: 'bundle.js'
    },
    devServer:{
      //端口号
      port:8080,
      //静态资源文件夹
      contentBase:'www'
    }
  };
(```)

## 虚拟DOM：用JavaScript对象描述DOM的层次结构。DOM中的一切属性都在虚拟DOM中有对应的属性

# diff是发生再虚拟DOM上的
## 新虚拟DOM和老虚拟DOM进行diff（精细化比较），算出应该如何最小量更新，最后反应到真正的DOM上
